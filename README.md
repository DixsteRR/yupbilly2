# 🤖 AI Telegram Bot — Инструкция запуска

## Что умеет бот

- 💬 **AI-ассистент** на базе GPT-4o-mini (умный и дешёвый)
- 🆓 **5 бесплатных сообщений** для новых пользователей
- 🎁 **3 дня пробного периода** — по кнопке
- 💳 **Платная подписка** через Telegram Stars (месяц / год)
- 🔄 **История чата** — бот помнит контекст разговора
- 📊 **Админ-панель** — статистика, рассылка

---

## ШАГ 1 — Создай бота в Telegram

1. Напиши [@BotFather](https://t.me/BotFather)
2. Команда `/newbot`
3. Придумай имя (например: `MyAI Assistant`)
4. Придумай username (например: `myai_helper_bot`)
5. Скопируй **токен** — он выглядит так: `1234567890:ABCDef...`

---

## ШАГ 2 — Получи ключ OpenAI

1. Зайди на [platform.openai.com](https://platform.openai.com)
2. Зарегистрируйся
3. API Keys → **Create new secret key**
4. Скопируй ключ (начинается с `sk-proj-...`)
5. Пополни баланс на $5–10 (хватит на тысячи сообщений)

> 💡 GPT-4o-mini стоит ~$0.00015 за сообщение — очень дёшево

---

## ШАГ 3 — Узнай свой Telegram ID

1. Напиши [@userinfobot](https://t.me/userinfobot)
2. Он ответит твоим ID, например: `123456789`
3. Это нужно для доступа к админ-командам

---

## ШАГ 4 — Деплой на Railway.app (бесплатно)

### 4.1 Подготовка
1. Создай аккаунт на [railway.app](https://railway.app)
2. Создай новый проект → **Deploy from GitHub**
3. Загрузи файлы бота в GitHub репозиторий

### 4.2 Структура файлов в репозитории
```
├── bot.py
├── database.py
├── requirements.txt
└── Procfile          ← создай этот файл!
```

### 4.3 Создай файл `Procfile`
```
worker: python bot.py
```

### 4.4 Переменные окружения на Railway
В панели проекта → **Variables** → добавь:

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | Токен от BotFather |
| `OPENAI_API_KEY` | Ключ OpenAI |
| `ADMIN_IDS` | Твой Telegram ID |

### 4.5 Деплой
Нажми **Deploy** — Railway сам установит зависимости и запустит бота!

## ШАГ 5 — Проверка

1. Найди своего бота в Telegram
2. Напиши `/start`
3. Проверь что бот отвечает
4. Напиши `/stats` чтобы проверить доступ админа

## Монетизация через Telegram Stars

Stars — встроенная валюта Telegram. Пользователи покупают их в приложении и платят ими напрямую тебе.

**Текущие тарифы в боте:**
- Месяц — 299 Stars (~$3.3)
- Год — 2490 Stars (~$27)

Вывод Stars: Telegram → Настройки → Stars → Вывод средств

---

## Административные команды

| Команда | Описание |
|---|---|
| `/stats` | Статистика: пользователи, подписки, сообщения |
| `/broadcast текст` | Рассылка всем пользователям |

---

## Изменение настроек (в bot.py)

```python
FREE_LIMIT   = 5      # бесплатных сообщений (можно поставить 3 или 10)
TRIAL_DAYS   = 3      # дней пробного периода
PRICE_MONTH  = 299    # Stars за месяц
PRICE_YEAR   = 2490   # Stars за год
```

---

## Примерная экономика

| Метрика | Значение |
|---|---|
| Стоимость 1 сообщения GPT-4o-mini | ~$0.00015 |
| 1000 сообщений в день | ~$0.15/день = ~$4.5/месяц |
| Цена подписки (месяц) | ~$3.3 (299 Stars) |
| Нужно подписчиков для покрытия | 2–3 платных пользователя |
| При 50 подписчиках | ~$150–165/мес чистыми |
| При 200 подписчиках | ~$600–660/мес чистыми |
